# Cache-Simulator
Cache Simulator for direct mapping, set associative and fully associative cache.
Expirements:
1- Hit ratio change with different block sizes, (Direct mapped).
2- Access time change with different number of cache levels, (Direct mapped).
3- Hit ratio change with different cache sizes, (Fully associative).
4- Hit ratio change with different replacement policies, (Fully associative).
5- Hit ratio change with different number of ways, (Set Associative).
6- Access time change with different number of cache levels, (Set associative).
